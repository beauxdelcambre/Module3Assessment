     The internet works by sending requests to servers, once the server receives the request it sends over the HTML for the page. After the HTML for the page is recieved the browser will render the HTML for the user to see on the webpage. The internet takes requests such as GET, POST, PUT, and DELETE, and responses such as HTML, JavaScript, CSS, and PNG/JPEG images!
     When you click a link your internet browser and operating system work together to get you to the website you are trying to reach. Once you click on the link the hostname of the webiste is turned into an IP address, this process is called the Domain Name Ststem (DNS). After, parts of the URL such as the resource, and the query argmuents are taken in to affect to ensure that you are taken to the correct page of the website.
